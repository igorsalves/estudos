# 📚 Central de Estudos

Bem-vindo ao meu repositório de estudos! Este repositório foi criado para organizar e documentar meu aprendizado em diversas plataformas de ensino. Aqui você encontrará cursos, desafios e projetos categorizados de forma clara e objetiva.

---

## 📂 Estrutura do Repositório

A organização do repositório segue a seguinte estrutura:

```
/estudos
│── /plataforma-a
│   ├── /curso-1
│   │   ├── README.md  # Descrição do curso e progresso
│   │   ├── /desafio-1
│   │   ├── /desafio-2
│   │   └── ...
│   ├── /curso-2
│   └── ...
│
│── /plataforma-b
│   ├── /curso-1
│   ├── /curso-2
│   └── ...
│
│── README.md  # Explicação sobre o repositório e estrutura
```

## 📌 Detalhamento

- **Plataformas**: Diretórios principais representando diferentes plataformas de ensino (exemplo: alura/, rocketseat/, udemy/, dio/ etc.).
- **Cursos**: Cada curso dentro de uma plataforma tem seu próprio diretório.
- **Desafios**: Dentro dos cursos, há subdiretórios para desafios e projetos práticos.

---

## 🚀 Como Utilizar

1. Clone o repositório:

```shell
  git clone https://github.com/seu-usuario/estudos.git
```

2. Navegue até a plataforma e curso desejado.

3. Leia os arquivos README.md para entender o conteúdo de cada curso/desafio.

4. Contribua adicionando anotações e melhorias ao seu aprendizado.
